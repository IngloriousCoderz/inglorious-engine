IngloriousScript in action!

IngloriousScript is as superset of JavaScript that allows you to define vectors and perform operations on them: 

      
const position = v(10, 20, 30)
const velocity = v(1, 1.5, 2)
const deltaTime = 0.16
const worldSize = 20

const newPosition = (position + velocity * deltaTime) % worldSize

Vectors are just plain number arrays, but with a special property that tells them apart: 

      
const position = v(10, 20, 30)
const [x, y, z] = position
position.__isVector__

Even though vectors are basically just number arrays, operations on plain number arrays will work the same as in JavaScript for compatibility: 

  
const position = [10, 20, 30]
const velocity = [1, 1.5, 2]
const deltaTime = 0.16
const worldSize = 20

const newPosition = (position + velocity * deltaTime) % worldSize
