# Circle

Intersection functions for a circle.

## Intersects Point

Checks if a circle intersects with a point.

<code>intersectsPoint(\{ position: [1, 0, 1], radius: 1 }, [1.5, 0, 1.5]) == {toString(intersectsPoint({ position: [1, 0, 1], radius: 1 }, [1.5, 0, 1.5]))}</code>&nbsp;


## Intersects Circle

Checks if a circle intersects with another circle.

<code>intersectsCircle(\{ position: [1, 0, 1], radius: 1 }, \{ position: [1, 0, 1], radius: 1 \}) == {toString(intersectsCircle({ position: [1, 0, 1], radius: 1 }, { position: [1, 0, 1], radius: 1 }))}</code>&nbsp;

<code>intersectsCircle(\{ position: [1, 0, 1], radius: 1 }, \{ position: [2, 0, 1], radius: 1 \}) == {toString(intersectsCircle({ position: [1, 0, 1], radius: 1 }, { position: [2, 0, 1], radius: 1 }))}</code>&nbsp;

<code>intersectsCircle(\{ position: [1, 0, 1], radius: 1 }, \{ position: [3, 0, 1], radius: 1 \}) == {toString(intersectsCircle({ position: [1, 0, 1], radius: 1 }, { position: [3, 0, 1], radius: 1 }))}</code>&nbsp;

<code>intersectsCircle(\{ position: [1, 0, 1], radius: 1 }, \{ position: [4, 0, 1], radius: 1 \}) == {toString(intersectsCircle({ position: [1, 0, 1], radius: 1 }, { position: [4, 0, 1], radius: 1 }))}</code>&nbsp;
