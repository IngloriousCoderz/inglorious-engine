# Geometry

Basic geometry functions.

## Hypothenuse

Uses Pythagora's theorem to calculate the hypothenuse of a right triangle (alias `pythagoras`).

<code>hypothenuse(3, 4) == {hypothenuse(3, 4)}</code>&nbsp;

## Pythagoras

Alias of `hypothenuse`.

<code>pythagoras(3, 4) == {pythagoras(3, 4)}</code>&nbsp;

## Is Point In Circle

Checks if a point is inside of a circle.

<code>isPointInCircle([1.5, 0, 1.5], \{ position: [1, 0, 1], radius: 1 \}) == {toString(isPointInCircle([1.5, 0, 1.5], { position: [1, 0, 1], radius: 1 }))}</code>&nbsp;

<code>isPointInCircle([2, 0, 2], \{ position: [1, 0, 1], radius: 1 \}) == {toString(isPointInCircle([2, 0, 2], { position: [1, 0, 1], radius: 1 }))}</code>&nbsp;

## Is Point In Rectangle

Checks if a point is inside of a circle.

<code>isPointInRectangle([1.5, 0, 1.5], \{ position: [1, 0, 1], size: [2, 2] \}) == {toString(isPointInRectangle([1.5, 0, 1.5], { position: [1, 0, 1], size: [2, 2] }))}</code>&nbsp;

<code>isPointInRectangle([2.5, 0, 2.5], \{ position: [1, 0, 1], size: [2, 2] \}) == {toString(isPointInRectangle([2.5, 0, 2.5], { position: [1, 0, 1], size: [2, 2] }))}</code>&nbsp;
