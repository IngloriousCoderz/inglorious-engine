# 2D

Functions to convert any vector to 2D and viceversa.

## From 2D

Builds a 3D vector from a 2D one.

<p>
  <code>from2D([3, 4]) == {toString(from2D([3, 4]))}</code>
</p>

## To 2D

Builds a 2D vector from a 3D one.

<p>
  <code>to2D([3, 0, 4]) == {toString(to2D([3, 0, 4]))}</code>
</p>
