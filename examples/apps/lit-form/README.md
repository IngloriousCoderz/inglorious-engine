# lit-form

Small demo showing the `form` type and helper components (inputs/selects/radios) built on top of `@inglorious/lit`.

Quick start:

```bash
cd examples/apps/lit-form
pnpm install
pnpm dev
```

Files of interest:

- `src/user-form/user-form.js` — shows how the `userForm` type is extended and handles submit/validation
- `src/user-form/fields/*` — field helper components (input, select, radio)
