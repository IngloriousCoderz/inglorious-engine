# Rectangle

Intersection functions for a rectangle.

## Intersects Point

Checks if a rectangle intersects with a point.

<code>intersectsPoint(\{ position: [1, 1, 0], size: [2, 2, 0] }, [1.5, 1.5, 0]) == {toString(intersectsPoint({ position: [1, 1, 0], size: [2, 2, 0] }, [1.5, 1.5, 0]))}</code>&nbsp;


## Intersects Circle

Checks if a rectangle intersects with a circle.

<code>intersectsCircle(\{ position: [1, 1, 0], size: [2, 2, 0] }, \{ position: [1, 1, 0], radius: 1 \}) == {toString(intersectsCircle({ position: [1, 1, 0], size: [2, 2, 0] }, { position: [1, 1, 0], radius: 1 }))}</code>&nbsp;

<code>intersectsCircle(\{ position: [1, 1, 0], size: [2, 2, 0] }, \{ position: [2, 1, 0], radius: 1 \}) == {toString(intersectsCircle({ position: [1, 1, 0], size: [2, 2, 0] }, { position: [2, 1, 0], radius: 1 }))}</code>&nbsp;

<code>intersectsCircle(\{ position: [1, 1, 0], size: [2, 2, 0] }, \{ position: [3, 1, 0], radius: 1 \}) == {toString(intersectsCircle({ position: [1, 1, 0], size: [2, 2, 0] }, { position: [3, 1, 0], radius: 1 }))}</code>&nbsp;

<code>intersectsCircle(\{ position: [1, 1, 0], size: [2, 2, 0] }, \{ position: [4, 1, 0], radius: 1 \}) == {toString(intersectsCircle({ position: [1, 1, 0], size: [2, 2, 0] }, { position: [4, 1, 0], radius: 1 }))}</code>&nbsp;

## Intersects Rectangle

Checks if a rectangle intersects with another rectangle.

<code>intersectsRectangle(\{ position: [1, 1, 0], size: [2, 2, 0] }, \{ position: [-1, -1, 0], size: [4, 4, 0] \}) == {toString(intersectsRectangle({ position: [1, 1, 0], size: [2, 2, 0] }, { position: [-1, -1, 0], size: [4, 4, 0] }))}</code>&nbsp;

<code>intersectsRectangle(\{ position: [1, 1, 0], size: [2, 2, 0] }, \{ position: [-2, 0, 0], size: [2, 2, 0] \}) == {toString(intersectsRectangle({ position: [1, 1, 0], size: [2, 2, 0] }, { position: [-2, 0, 0], size: [2, 2, 0] }))}</code>&nbsp;

<code>intersectsRectangle(\{ position: [1, 1, 0], size: [2, 2, 0] }, \{ position: [-1, -1, 0], size: [2, 2, 0] \}) == {toString(intersectsRectangle({ position: [1, 1, 0], size: [2, 2, 0] }, { position: [-1, -1, 0], size: [2, 2, 0] }))}</code>&nbsp;

<code>intersectsRectangle(\{ position: [1, 1, 0], size: [2, 2, 0] }, \{ position: [-3, 0, 0], size: [2, 2, 0] \}) == {toString(intersectsRectangle({ position: [1, 1, 0], size: [2, 2, 0] }, { position: [-3, 0, 0], size: [2, 2, 0] }))}</code>&nbsp;
